# Peer Review — Story 00000012: Toggle for the two-player board flip

## Summary

This branch adds a player-facing "Flip board between turns" toggle that, when
off, always draws the Phase 2 board from red's perspective. It threads a
`flipBetweenTurns` flag through `viewSide` and `PlayBoard`, adds a best-effort
local-storage persistence helper, a new `FlipBoardToggle` component, and a
README mention. `npm run typecheck`, `npm run lint`, and `npm test` (355 tests,
19 files) all pass. The logic (the `viewSide` seam, persistence, wiring) closely
follows the plan, but the new toggle's stylesheet is never wired into the module
graph, so its intended switch styling is not applied.
